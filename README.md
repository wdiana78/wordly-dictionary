# Wordly Dictionary

## Description

Wordly Dictionary is a Single Page Application (SPA) that allows users to search for English words and retrieve their meanings using the Free Dictionary API.

## Features

- Search for words
- Display pronunciation
- Display definitions
- Display examples
- Display synonyms
- Play pronunciation audio
- Error handling
- Responsive interface

## Technologies

- HTML
- CSS
- JavaScript
- Fetch API
